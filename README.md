# Tabla-Busuqeda-PHP-Jquery-MySql
 Tabla que se renderiza en misma pagina con conexion a base de datos
 
 Los siguientes codigos, muestran archivos web PHP que muestran una consulta a una base de dators, renderizando una tabla, (la cual se rellena con la consulta echa a una base de datos).  Esta tabla cuenta tanto como buscar dentro de lla u organizar las columas en orden ascendiente o descendiente.
